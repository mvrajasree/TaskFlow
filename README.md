# TaskFlow

A full-stack task management app built with React, Node.js, Express, and PostgreSQL.

🔗 Live Demo: https://task-flow-lemon-nine-73.vercel.app/

## Tech Stack

- **Frontend:** React, Tailwind CSS, Axios
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **Deployment:** Vercel (frontend), Railway (backend + database)

## Features

- Create, read, update, delete tasks
- Priority levels (High / Medium / Low)
- Mark tasks as complete
- Due date tracking
- Toast notifications
- Responsive UI

## Local Setup

### Backend
cd backend
npm install
# Create .env file with your PostgreSQL credentials
npm run dev

### Frontend
cd frontend
npm install
npm run dev
